select, optgroup, option
여러 옵션 중에서 선택 - 드롭다운 목록
공간을 최소한으로 사용하면서 여러 옵션 표시 가능
select 태그의 속성
size : 화면에 표시될 드롭다운 메뉴의 항목 개수르 지정
크롬 브라우저의 경우,size에서 지정한 개수보다 하나 더많은 옵션이 표시
multiple : 브라우저 화면에 여러 개의 옵션이 함께 표시되면서 ctrl키를 누른 상태로 드롭다운 메뉴에 잇는 여러 항목을 선택할 수 있다
option 태그의 속성
value : 옵션을 선택했을 때 서버로 넘겨질 값을 지정
selected : 화면에 표시될 때 기본으로 선택되어 있는 옵션을 지정
