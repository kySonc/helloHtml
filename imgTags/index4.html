area 태그, usemap 속성 - 이미지 맵
이미지맴 : 한 이미지 상에 여라 다른 링크를 만드는 것
coords : 링크로 사용할 영역을 시작 좌표와 끝 좌표를 이용해 지정
media : 링크 문서(사이트)를 어떤 미디어에 최적화 시킬지 지정
shape : 링크로 사용할 영역의 형태를 지정 <속성값> default, rect, circle, paly
target : 링크로 사용할 영역의 형태를 지정
type : 링크 문서의 미디어 유형을 지정
alt : 대체 텍스트를 지정
href : 링크 문서(사이트) 경로를 지정 <속성값> bookmark, help, tag, next,....
