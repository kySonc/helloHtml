텍스트

a 태그에서 사용할 수 있는 속성>
href : 링크한 문서나 사이트의 주소를 입력
target : 링크한 내용이 표시될 위치(현재 창 또는 새창)을 지정
download : 다운로드를 한다
rel : 현재문서와 링크한 문서의 관계를 알려준다
hreflang : 링크한 문서의 언어를 지정한다
type : 링크한 문서의 파일 유형을 알려준다

target 의 사용할 수 있는 속성값
_blank : 링크 내용이 새 창이나 새 탭에서 열린다
_self : target 속성의 기본값으로 링크가 화면에서 열린다
_parent : 프레임을 사용했을 때 링크 내용을 부모 프레임에 표시한다.
_top : 프레임을 사용했을 때 프레임을 벗어나 링크 내용을 전체화면에 표시
